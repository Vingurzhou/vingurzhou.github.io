import random
length = random.randint(1, 10)
print('--------他的长短---------')
guess = int(input("他多长?"))
if guess == length:
        print("你怎么知道.")
        print("你是不是见过.")
else:
        while guess != length:
                temp = input("再试一次.")
                guess = int(temp)
                if guess > length:
                        print("他没那么长")
                elif guess < length:
                        print("他没那么短")
                else:
                        print("你怎么知道")
                        print("你是不是见过")
print("结束.")

流程

1

流程:就是计算机执行代码的顺序
流程控制:对计算机代码执行的顺序进行有效的管理,只有流程控制才能实现在开发当中的业务

流程控制的分类
顺序流程:就是代码一种自上而下的执行结构,也是 python默认的
选择流程/分支流程:根据在某一步的判断,有选择的去执行相应的逻辑的一种结构|
循环流程:满足一定的条件下,一直重复的去执行某段代码的逻辑[事情]
