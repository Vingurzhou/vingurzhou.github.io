random

import random
length = random.randint(1, 10)
print('--------他的长短---------')
guess = int(input("他多长?"))
if guess == length:
        print("你怎么知道.")
        print("你是不是见过.")
else:
        while guess != length:
                temp = input("再试一次.")
                guess = int(temp)
                if guess > length:
                        print("他没那么长")
                elif guess < length:
                        print("他没那么短")
                else:
                        print("你怎么知道")
                        print("你是不是见过")
print("结束.")

流程

1

流程:就是计算机执行代码的顺序
流程控制:对计算机代码执行的顺序进行有效的管理,只有流程控制才能实现在开发当中的业务

流程控制的分类
顺序流程:就是代码一种自上而下的执行结构,也是 python默认的
选择流程/分支流程:根据在某一步的判断,有选择的去执行相应的逻辑的一种结构|
循环流程:满足一定的条件下,一直重复的去执行某段代码的逻辑[事情]

循环

1

import random
count = 1
while count <= 3:
        person = int(input('出'))
        computer = random.randint(0, 2)
        print(computer)
        if person == computer:
                print('平')
        elif person == 0 and computer == 1 or person == 1 and computer == 2 or person == 2 and computer == 0:
                print('赢')
        else:
                print('输')
        count += 1
print('结束')

introduction:数字和数字比较,不能随意给字符串赋值;缩进;random必须在while内部.

字符串及常用方法

序列: 一组按照顺序排列的值[数据集合]
内置的序列类型: 字符串、列表、元组
序列优点: 可以支持索引和切片的操作
序列特征: 第一个正索引为0,指向的是左端,第一个索引为复数的时候,指向的是右端

切片: [高级特性]可以根据下标来获取序列对象的任意数据
语法结构: [start:end:step] step默认1
