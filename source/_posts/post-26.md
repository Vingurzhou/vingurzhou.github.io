## 第八章 排序

|              | 最好         | 最坏    | 逻辑                   | 特性         |
| ------------ | ------------ | ------- | ---------------------- | ------------ |
| 插入排序     | $$n$$        | $$n^2$$ | 插入到排序好的序列内   | 基本有序最快 |
| 冒泡排序     | $$n$$        | $$n^2$$ | 相邻交换               | 交换最多     |
| 快速排序     | $$\log_2 n$$ | $$n^2$$ |                        | 平均最快     |
| 简单选择排序 | $$n^2$$      | $$n^2$$ | 遍历找最小跟第一个交换 | 交换最少     |
| 希尔排序     |              |         | gap=n/2                |              |
| 基数排序     |              |         | 个十百                 |              |
| 归并排序     |              |         | 合并有序数组           |              |
